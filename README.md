# Non-ASCII Character Checker

A React web application that identifies and highlights non-ASCII characters in text strings, making it easy to spot problematic characters in large text files or data imports.

## Usage

1. Paste or type your text into the textarea
2. The app automatically analyzes the text as you type
3. Non-ASCII characters are highlighted in yellow
4. View the count and list of unique non-ASCII characters at the top of the results
